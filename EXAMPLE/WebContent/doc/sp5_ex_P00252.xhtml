[SelectBox] DataList와 바인딩하여 선택 항목을 설정하기

1개요

WebSquare툴 내부에서 DataList 작성법과 SelectBox컴포넌트에 DataList를 바인딩하여 선택 항목을 설정하는 예제입니다.

이 기능은 아래의 속성과 함수로 사용할 수 있습니다.
- appearance : (속성)선택항목 표현방식
- setNodeSet : (함수)SelectBox컴포넌트에 DataList 바인딩
- addItem : (함수)선택 항목을 추가. index를 지정하지 않으면 하위로 추가됨.

P00252

  • 함수로 DataList 바인딩 항목 늘리기
