[GridView] Event - oncellclick (셀을 클릭할 때 발생)

1개요

GridView의 이벤트 onCellClick 예제입니다.
onCellClick 이벤트는 Body영역의 셀이 클릭된 경우 발생합니다.
지정된 이벤트 핸들러에서 클릭된 셀의 row index, column index, column id 값을 확인할 수 있습니다.

P00217

2구현된 기능

  • 이벤트 oncellclick 발생 시 로그 출력하기

3예제 테스트 방법

3.1이벤트 oncellclick 발생 시 로그 출력하기

STEP1. 컬럼 '도서명'의 1번째 행의 셀을 클릭합니다.

영역 [GridView의 oncellclick 이벤트 핸들러 등록]의 GridView를 확인합니다.

컬럼 '도서명'의 1번째 행의 셀을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 실행 결과를 확인합니다.

oncellclick 이벤트가 발생되고 이벤트 핸들러가 실행되어 로그가 출력됩니다.

영역 [로그 확인] 또는 브라우저의 개발자 도구의 콘솔(console)탭에 출력된 로그를 확인합니다.

[브라우저(Chrome) 실행 예시 - 로그 확인]

[로그 예시]

[05:51:21] ** scwin.grd_exam1_oncellclick **

row index : 0

column index : 0

column id : book_name

4구현 예시

4.1이벤트 oncellclick의 핸들러 지정하기

STEP1. GridView의 이벤트 oncellclick 핸들러를 지정합니다.

예제 파일에서는 핸들러로 사용할 함수명을 scwin.grd_exam1_oncellclick로 정의하였습니다.

그림 0-1.웹스퀘어5 SP5 스튜디오의 Property View(속성창)의 이벤트 탭 예시

[소스 코드 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView ev:oncellclick="scwin.grd_exam1_oncellclick" id="grd_exam1">
    <!-- 중략 -->
</w2:gridView>

STEP2. 핸들러 함수 scwin.grd_exam1_oncellclick를 정의합니다.

[소스 코드 예시]

    switch (colId) {
        case "book_name":

        default:
