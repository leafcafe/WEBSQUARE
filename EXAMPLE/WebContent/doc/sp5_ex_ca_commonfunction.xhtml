04.컴포넌트 공통 기능

컴포넌트에 공통으로 적용된 API의 예제들입니다. 특수 컴포넌트를 제외한 대부분의 컴포넌트에서 지원합니다.

  • [컴포넌트 공통] 컴포넌트의 이름(pluginName) 반환받기
  • [컴포넌트 공통] dataset 기능 - userData
  • [컴포넌트 공통] 스크립트로 style 제어하기
  • [컴포넌트 공통] 스크립트로 class 제어하기
  • [컴포넌트 공통] 스크립트로 이벤트 연결(추가)하기
  • [컴포넌트 공통] 스크립트로 이벤트 해제(제거)하기
  • [컴포넌트 공통] 스크립트로 이벤트 발생시키기
  • [컴포넌트 공통] 클라이언트 다국어 적용하기 - localeRef
  • [컴포넌트 공통] 클라이언트 다국어 적용하기(목록) - useItemLocale
  • [컴포넌트 공통] 클라이언트 다국어 적용하기(툴팁) - tooltipLocaleRef
  • [컴포넌트 공통] 클라이언트 다국어 KEY가 등록되지 않는 경우 label, value 출력하기 - useDefaultLabel
