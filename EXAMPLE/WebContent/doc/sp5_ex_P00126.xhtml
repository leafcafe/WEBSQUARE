[Input] 입력값의 좌우 공백 제거하기 - trim, ltrm, rtrm

1개요

Input에 입력한 데이터의 좌우 공백 제거 예제입니다.
속성 trim, ltrim, rtrim 설정으로 기능을 사용할 수 있습니다.
이 기능은 컴포넌트에서 입력 커서가 벗어날 때 적용됩니다.

P00126

2구현된 기능

  • 기본 설정 - 입력값 공백 제거 안함

  • 입력값 좌우 공백 제거

  • 입력값 좌측 공백 제거

  • 입력값 우측 공백 제거

3예제 테스트 방법

3.1기본 설정 - 입력값 공백 제거 안함

STEP1. 영역 [기본 설정]의 Input에 값 " 123 456 "을 입력합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 컴포넌트의 외부 영역을 클릭하여 커서를 제거합니다.

[브라우저(Chrome) 실행 예시]

STEP3. 실행 결과를 확인합니다.

Input의 입력 영역을 클릭하여 입력값을 확인합니다.

입력한 값이 " 123 456 "으로 동일한 것을 확인합니다.

[브라우저(Chrome) 실행 예시]
