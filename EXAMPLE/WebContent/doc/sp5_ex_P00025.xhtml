[Group] 자손 컴포넌트의 Method 일괄 호출하기

1개요

Group 컴포넌트의 invoke method 예제입니다. Group의 자손 컴포넌트의 method를 일괄 호출하는 기능을 제공합니다. 자손 컴포넌트 중 실행할 method가 존재하지 않는 경우는 오류 없이 제외하고 진행합니다.

P00025

2구현된 기능

  • Group 컴포넌트의 전체 자손 컴포넌트에 setValue 함수 호출하기

  • Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 disabeld하기

  • Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 enabeld하기

3예제 테스트 방법

Group 컴포넌트의 전체 자손 컴포넌트에 setValue 함수 호출하기

1. 버튼 실행1을 클릭합니다.

2. 예제 화면의 InputBox, CheckBox, InputCalendar, TextBox 영역의 컴포넌트의 value가 빈 문자열로 할당됩니다.

Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 disabeld하기

1. 버튼 실행2를 클릭합니다.

2. 예제 화면의 InputBox, CheckBox, InputCalendar 영역의 컴포넌트가 비활성화됩니다.

Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 enabeld하기

0. 버튼 실행2를 클릭하여 컴포넌트를 비활성화 한 뒤 진행해야 컴포넌트의 변화를 확인 할 수 있습니다.

1. 버튼 실행3을 클릭합니다.

2. 예제 화면의 InputBox, CheckBox, InputCalendar 영역의 컴포넌트가 활성화됩니다.

4구현 예시

4.1Group 컴포넌트의 전체 자손 컴포넌트에 setValue 함수 호출하기

//grp_form 컴포넌트 하위의 모든 웹스퀘어 컴포넌트의 setValue method를 실행합니다.
//setValue method가 없는 경우 해당 컴포넌트는 제외하고 진행됩니다.
grp_form.invoke("setValue","");

4.2Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 disabeld하기

//grp_form 컴포넌트 하위의 모든 웹스퀘어 컴포넌트의 setDisabled method를 실행합니다.
//setDisabled method가 없는 경우 해당 컴포넌트는 제외하고 진행됩니다.
//grp_form 컴포넌트 하위의 컴포넌트가 disabeld됩니다.
grp_form.invoke("setDisabled",true);

4.3Group 컴포넌트의 전체 자손 컴포넌트를 setDisabled 함수로 enabeld하기

//grp_form 컴포넌트 하위의 모든 웹스퀘어 컴포넌트의 setDisabled method를 실행합니다.
//setDisabled method가 없는 경우 해당 컴포넌트는 제외하고 진행됩니다.
//grp_form 컴포넌트 하위의 컴포넌트가 enabled됩니다.
grp_form.invoke("setDisabled", false);

5주요 API

  • invoke( funcName , param , options )
