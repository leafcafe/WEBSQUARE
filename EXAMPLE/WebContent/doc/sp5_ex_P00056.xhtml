wFrame 팝업을 지정한 WFrame 영역에 표시하기

1개요

wFrame 팝업의 이동 영역과 Modal 영역을 지정하는 예제입니다.
이 기능은 $p.openPopup API의 옵션 중 "frameModal" 속성을 사용합니다.
이 예제는 팝업의 유형이 wframePopup인 경우에만 해당됩니다.

P00056

2구현된 기능

  • 속성 frameModal을 frame으로 할당하여 실행하기

  • 속성 frameModal을 parent으로 할당하여 실행하기

  • 속성 frameModal을 top으로 할당하여 실행하기

  • 속성 frameModal을 특정 WFrame ID로 지정하여 실행하기

3예제 테스트 방법

예제는 총 4개의 화면 파일로 구성되어있고 파일별 대략적인 구조는 아래와 같습니다.

  • P00056.xml : 예제 실행 파일로 wframe이 구성되어있고 wframe의 src는 P00056S01.xml 입니다.

  • P00056S01.xml : wframe이 구성되어있고 wframe의 src는 P00056S02.xml 입니다.

  • P00056S02.xml : wframe이 구성되어있고 wframe의 src는 P00056S03.xml 입니다.

  • P00056S03.xml : wFrame 팝업을 실행하는 스크립트가 작성되어있습니다.

그림 0-1.예제 실행 예시

wframe의 영역의 구분을 테두리로 표시하였습니다.

화면의 버튼을 클릭하여 팝업이 뜨는 위치, 이동 영역, Modal 영역을 확인합니다.

속성 frameModal을 frame으로 할당하여 실행하기

버튼 frameModal : "frame"을 클릭합니다.

실행된 팝업은 스크립트를 실행한 P00056S03.xml 영역에서만 이동이 가능합니다.

속성 frameModal을 parent으로 할당하여 실행하기

버튼 frameModal : "parent"을 클릭합니다.

실행된 팝업은 스크립트를 실행한 P00056S03.xml의 부모 파일인 P00056S02.xml 영역에서만 이동이 가능합니다.

속성 frameModal을 top으로 할당하여 실행하기

버튼 frameModal : "top"을 클릭합니다.

실행된 팝업은 최상위 웹스퀘어 영역, 예제를 기준으로 브라우저 전체 영역에서 이동이 가능합니다.

속성 frameModal을 특정 WFrame ID로 지정하여 실행하기

버튼 frameModal : [Frame ID]을 클릭합니다.

실행된 팝업은 P00056.xml 영역에서만 이동이 가능합니다. frameModal 속성에 P00056.xml 영역의 Frame ID를 할당하였습니다.

4구현 예시

팝업 실행 스크립트는 예제 파일 P00056S03.xml에 작성되어 있습니다.

4.1frameModal 속성 할당하기 - frame, top, parent

frameModal의 속성값 중 frame, top, parent은 고정 문자열입니다.

//팝업 옵션
var jsnOptions = { 
    id : "popup_exam1", 
    type : "wframePopup", //필수 고정
    width: "200px", 
    height: "150px", 
    top: "0px", 
    left: "0px",
    popupName : "frameModal - frame", 
    modal : "true", //필수 고정
    frameModal : "frame" //필수 선택 - modal을 표시해줄 frame을 지정. "top", "parent", "frame", frame id
}; 

$p.openPopup("/page/P00056P01.xml",  jsnOptions );
이 예제와 동일한 기능을 구현하기 위해서는 아래의 옵션은 고정으로 할당해야 합니다.

type : "wframePopup"
modal : "true"

//목적에 따라 선택하여 할당
frameModal : ["top", "parent", "frame", frame id]

4.2frameModal 속성 할당하기 - Frame ID

Frame ID는 실행 시점에 생성된 ID를 지정해야합니다. 그렇기 때문에 고정값이 아닌 스크립트를 이용하여 ID를 반환받아 할당합니다.
var strFrameID;
var jsnOptions;

//컨텐츠 영역(P00056.xml)의 ID - 자식의 화면에서 부모의 Frame ID를 획득하기 위해서는 아래와 같이 상대적인 방법으로 접근해야 합니다.
strFrameID = $p.getFrame().getParentFrame().getParentFrame().getParentFrame().id

jsnOptions = { 
    id : "popup_exam4", 
    type : "wframePopup",
    width: "200px", 
    height: "150px", 
    top: "0px", 
    left: "0px",
    popupName : "frameModal - frameID", 
    modal : "true",
    frameModal : strFrameID //실행 시점에 생성된 Frame ID
};

$p.openPopup("/page/P00056P01.xml",  jsnOptions );
이 예제와 동일한 기능을 구현하기 위해서는 아래의 옵션은 고정으로 할당해야 합니다.

type : "wframePopup"
modal : "true"

//목적에 따라 선택하여 할당
frameModal : ["top", "parent", "frame", frame id]
