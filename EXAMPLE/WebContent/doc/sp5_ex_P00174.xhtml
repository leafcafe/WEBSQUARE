[GridView] inputType이 checkbox로 지정된 열의 선택, 선택 해제 값 지정하기

1개요

GridView의 컬럼 속성 inputType이 checkbox일 때, 
컬럼 속성 valueType(값의 유형)의 설정 값에 따라 엔진에서 할당되는 체크된 값과 체크가 해제된 값을 비교하는 예제입니다.
속성 valueType의 설정 값에 따라 엔진에서 checkbox가 체크되었을 때 value로 할당하는 값과 체크가 해제되었을 때 value로 할당하는 값이 달라집니다.
사용자(개발자)가 직접 값을 할당하고자 하는 경우는 valueType을 other로 지정하고 속성 trueValue(체크된 값)와 falseValue(체크 해제된 값)를 지정합니다.

P00174

checkbox와 연결된 DataList의 컬럼의 기본 값을 설정하는 자세한 예시는 아래의 링크를 통해 확인할 수 있습니다.
- 문서 링크 : [GridView] inputType이 checkbox로 지정된 열의 기본 값 지정하기
- 예제 화면 ID : P00173 (예제 사이트에서 검색 가능합니다)

2구현된 기능

  • valueType이 binary로 설정되었을 때 checkbox의 value 확인하기

  • valueType이 boolean으로 설정되었을 때 checkbox의 value 확인하기

  • valueType이 other로 설정되었을 때 checkbox의 value 확인하기

3예제 테스트 방법

3.1valueType이 binary로 설정되었을 때 checkbox의 value 확인하기

체크된 값 : 1, 체크 해제된 값 : 0, 기본 값(사용자 지정) : 0

4.1GridView의 컬럼 속성 valueType 지정하기 - binary, boolean

GridView의 컬럼 속성을 지정합니다.

  • [필수] valueTyp="옵션 값" //값의 유형을 설정합니다.

    ( 옵션 값 )

    binary : 체크된 값은 "1", 체크 해제된 값은 "0"으로 할당합니다.

    예시) defaultValue="binary"

    boolean : 체크된 값은 "true", 체크 해제된 값은 "false"로 할당합니다.

    예시) defaultValue="boolean"

<w2:gridView dataList="data:dlt_books_1" id="grd_exam1" style="height:100px;">

        <w2:row id="row2" style="">

</w2:gridView>

4.2GridView의 컬럼 속성 valueType 지정하기 - other

GridView의 컬럼 속성을 지정합니다.

  • [필수] valueTyp="other" //값의 유형을 설정합니다.

  • [필수] trueValue="설정 값" //체크된 값

    예시) trueValue="Y"

  • [필수] falseValue="설정 값" //체크 해제된 값

    예시) falseValue="N"
