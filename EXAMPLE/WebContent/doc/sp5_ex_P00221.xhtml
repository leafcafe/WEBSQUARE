[Accordion] 한 패널을 연 후에 다른 패널을 열 경우, 처음 연 패널(들)을 열린 채로 유지하기

1개요

패널 클릭 시 다른 패널의 열림/닫힘 상태에 영향을 주지 않는 속성에 관한 예제입니다.
이 기능은 아래의 속성으로 사용할 수 있습니다.
- independentPanel : 타 패널을 클릭 시 기존 패널의 오픈 상태 유지 또는 비유지

P00221

2구현된 기능

  • 속성 'independentPanel' 미적용

  • 속성 'independentPanel' 적용

3예제 테스트 방법

3.1속성 'independentPanel' 미적용

STEP1. 패널을 클릭합니다.

panel2를 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 결과 확인 및 패널 클릭합니다.

panel1이 닫히고 panel2가 열려있는 것을 확인하고 panel3을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP3. 실행 결과를 확인합니다.

panel2가 닫히고 panel3이 열려있는 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

3.2속성 'independentPanel' 적용

STEP1. 패널을 클릭합니다.

panel5를 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. 결과 확인 및 패널 클릭합니다.

panel4, panel5가 열려있는 것을 확인하고 panel6을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP3. 실행 결과를 확인합니다.

panel4, panel5, panel6이 열려있는 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1패널의 오픈 여부 지정하기

Accordion의 속성을 정의합니다.

  • [필수] independentPanel="true" // [default: "false", "true"] 패널 클릭시 다른 패널의 열림/닫힘 상태에 영향을 주지 않는 속성

그림 0-1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<!-- accordion 의 소스 본문 예시 -->
<w2:accordion independentPanel="true" id="acd_exam_2">
    <!-- 중략 -->
</w2:accordion>
