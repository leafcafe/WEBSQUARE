[GridView] Groupby 기능 적용하기

1개요

그룹핑 기능을 사용하면 한 개 이상의 컬럼을 기준으로 GridView의 데이터를 그룹으로 분류하고 
각 그룹에 대한 통계 값을 확인할 수 있습니다.
이 예제에서는 GridView의 ongroupby를 이용한 방법과 별도의 이벤트나 함수를 이용해 적용하는 방법, 이 두가지 방법을 모두 적용한 예제 입니다.
ongroupby는 드래그 동작이 필요해 모바일에는 사용할 수 없지만 이벤트나 함수를 활용한 방법으로 구현 가능합니다.

P00194

2구현된 기능

  • groupby를 실행해 Grouping하기

  • ongroupby 이벤트 활용해 Grouping 하기

  • clearGroup으로 Grouping 해제하기

3예제 테스트 방법

3.1Grouping 실행하기

STEP1. 초기 상태 확인하기

영역 [Grouping]의 GridView를 확인합니다.

GridView의 속성 Groupbyzone이 'true'로 지정되어 상단에 Groupbyzone이 나타납니다.

[브라우저(Chrome) 실행 예시]

STEP2. Groupby 실행하기

버튼 Groupby를 클릭합니다.

STEP3. 실행 결과를 확인합니다.

Groupby가 적용되어 통계 결과가 나오는 것을 확인할 수 있습니다.

[브라우저(Chrome) 실행 예시]

STEP4. Groupby 해제하기

버튼 ClearGroupby를 클릭합니다.

STEP5. 실행 결과를 확인합니다.

Groupby가 해제되어 원래의 GridView가 나오는 것을 확인할 수 있습니다.

[브라우저(Chrome) 실행 예시]

STEP6. Groupby 실행하기

Gridview의 컬럼을 GridView 상단의 groupbyzone에 Drag & Drop 합니다.

[브라우저(Chrome) 실행 예시]

STEP7. 실행 결과를 확인합니다.

Groupby가 적용되어 통계 결과가 나오는 것을 확인할 수 있습니다. 이미지는 "Country"와 "Item"을 순서대로 Drag & Drop 한 결과입니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1Grouping 적용하기

STEP1. Groupby를 실행시킬 컴포넌트에 이벤트를 정의합니다.

예제의 경우 컴포넌트 "Trigger"를 사용했으며 이벤트는 "onclick"을 사용했습니다.

[소스 코드 예시]

<!-- Grouping 의 소스 본문 예시 - Trigger -->
<xf:trigger class="" id="btn_groupby" style="" type="button" 
ev:onclick="scwin.btn_groupby_onclick">
    <xf:label><![CDATA[Groupby]]></xf:label>
</xf:trigger>

그림 0-1.웹스퀘어5 SP5 스튜디오의 Property View(속성창)의 이벤트 탭 예시

STEP2. 이벤트 함수 내부에 Groupby의 옵션을 정의하고 "GridView"의 "groupby()"실행합니다.

[소스 코드 예시]

scwin.btn_groupby_onclick = function(e) {
    // groupby에서 사용할 옵션 객체
    var options = {
        sortIndex: [0, 1],      // 기준 컬럼 index (0번, 1번)
        sortOrder : [1, 1],     // 컬럼의 정렬을 지정 (0: 내림차순, 1: 오름차순)
        closeGroup: true,       // groupby 된 데이터의 숨김 상태 여부
        groupbyHeader: [        // 전체 건수를 헤더에 표현하는 방식 설정
            {
                // expression 그룹핑 컬럼 데이터를 계산식으로 표현
                inputType: "expression",    
                // 표현식 : 토글버튼, 컬럼에 속한 값, 데이터 건수 표시
                expression: 'toggle() + depthStr() + "[ " + count() + " Cases ]"',
                align: "left", colSpan: "4" 
            }
        ],
        groupbyFooter: {    // 1단, 2단까지 표현 형식을 지정
            depth_0: [      //1단 Footer 내용
                { colSpan: "3", value: "Total", align: "right" },
                // col4의 총 합
                { inputType: "expression", expression: 'SUM("col4")' }
            ],
            depth_1: [      //2단 Footer 내용
                { colSpan: "3", value: "Sub Total", align: "right" },
                // col4의 총 합
                { inputType: "expression", expression: 'SUM("col4")' }
            ]
        }
    };
    grd.groupby(options);    // groupby 실행
};

4.2ongroupby 적용하기

STEP1. GridVIew의 groupbyZone 속성을 true로 지정합니다.

  • [필수] groupbyZone="true" // groupbyZone 활성화

그림 0-2.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

STEP2. GridVIew의 ongroupby 이벤트를 설정합니다.

GridView에 ongroupby이벤트를 "scwin.ongroupby"로 설정합니다.

[소스 코드 예시]

<!-- Grouping 의 소스 본문 예시 - GridView 설정 -->
<w2:gridView ... ev:ongroupby="scwin.ongroupby" groupbyZone="true"  ...>
// 중간 생략
</w2:gridView>
STEP3. 이벤트 함수 내부에 Groupby의 옵션을 정의하고 "GridView"의 "groupby()"실행합니다.

[소스 코드 예시]

scwin.ongroupby = function (options) {
    // groupby에서 사용할 옵션 객체
    options.closeGroup = true;      // groupby 된 데이터의 숨김 상태 여부
    options.groupbyHeader = [       // 전체 건수를 헤더에 표현하는 방식 설정
        {
            colSpan: "4",
            inputType: "expression",
            expression: 'toggle() + depthStr() + "[ " + count() + " Cases ]"',
            align: "left", colSpan: "4"
        }
    ];
    options.groupbyFooter = [       // 단계별 지정이 아닌 한가지로 표현
        { colSpan: "3", value: "Total", align: "right" },
        { colSpan: "1", inputType: "expression", expression: 'SUM("col4")' }
    ];
    grd.groupby(options); //그룹핑 시작
}

4.3Groupby 해제하기

STEP5. "clearGroupby"를 실행시킬 컴포넌트에 이벤트를 정의합니다.

예제의 경우 컴포넌트 "Trigger"를 사용했으며 이벤트는 "onclick"을 사용했습니다.

[소스 코드 예시]

<!-- Grouping 의 소스 본문 예시 - Trigger -->
<xf:trigger class="" id="btn_clear" style="" type="button" 
ev:onclick="scwin.btn_clear_onclick">
    <xf:label><![CDATA[Groupby]]></xf:label>
</xf:trigger>

그림 0-3.웹스퀘어5 SP5 스튜디오의 Property View(속성창)의 이벤트 탭 예시

STEP6. 이벤트 함수 내부에 GridView의 "clearGroupby"를 실행합니다.

[소스 코드 예시]

scwin.btn_clear_onclick = function(e) {
    grd.clearGroupby();
};
