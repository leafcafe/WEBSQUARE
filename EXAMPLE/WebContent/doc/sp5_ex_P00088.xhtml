[GridView] 데이터 복사 - 데이터 출처 설정

1개요

GridView에서 데이터 복사(Ctrl+C)를 할 때 데이터 출처의 설정 예제입니다.

기본 동작은 GridView와 연결된 DataList의 값으로 복사되는 것이며, 설정을 통해 화면에 출력된 값을 복사하도록 변경할 수 있습니다.

P00088

2구현된 기능

  • 복사할 데이터의 출처를 GridView 연결된 DataList의 값으로 설정하기

  • 복사할 데이터의 출처를 GridView에 출력된 값으로 설정하기

  • GridView의 컬럼별 복사할 데이터 출처 설정하기

3예제 테스트 방법

이 예제의 GridView는 같은 DataList와 연결되어있습니다.
DataList의 값과 화면에 출력된 값이 다른 컬럼은 정가, 출간일 입니다.
각 컬럼의 데이터를 복사(Ctrl+C)/붙여넣기(Ctrl+V)하여 복사된 값을 비교합니다.
이 예제의 데이터 복사는 키보드의 단축키(Ctrl+C)를 눌러 사용할 수 있으며, 셀(Cell) 단위로 동작합니다.
STEP1. 복사할 셀(Cell)을 선택(클릭)합니다.

[실행된 화면 예시 이미지]

STEP2. 데이터를 복사합니다.

키보드 복사 단축키 Ctrl + C를 누릅니다.

STEP3. 복사된 데이터를 확인합니다.

메모장, 엑셀 등의 에디터에서 붙여넣기(Ctrl + V)하여 복사된 데이터를 확인합니다.

DataList의 값이 복사된 경우는 16600 이 복사됩니다.

화면에 출력된 값이 복사된 경우는 16,500 이 복사됩니다.

4구현 예시

DataList 생성 및 연결은 생략되었습니다.

4.1복사할 데이터의 출처를 화면에 출력된 값으로 설정하기 - GridView 전체 설정

GridView의 속성을 정의합니다.

  • copyOption="display" //데이터 복사 시 화면에 출력된 값을 기준으로 복사합니다.

    "dataList" (기본 값) : 데이터 복사 시 dataList에 들어있는 값을 기준으로 복사합니다.

    "display" : 데이터 복사 시 화면에 출력된 값을 기준으로 복사합니다.

STEP1. 디자인탭에서 GridView를 선택합니다.

[웹스퀘어5 SP5 스튜디오의 Design 탭 예시]

STEP2. GridView의 속성을 설정합니다.

[웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView copyOption="display" dataList="data:dlt_books" >
    <!-- 중략 -->
</w2:gridView>

4.2복사할 데이터의 출처를 화면에 출력된 값으로 설정하기 - 컬럼별 설정

적용할 컬럼의 속성을 정의합니다.

  • copyOption="display" //데이터 복사 시 화면에 출력된 값을 기준으로 복사합니다.

    "dataList" (기본 값) : 데이터 복사 시 dataList에 들어있는 값을 기준으로 복사합니다.

    "display" : 데이터 복사 시 화면에 출력된 값을 기준으로 복사합니다.

STEP1. 디자인탭에서 GridView의 컬럼를 선택합니다.

[웹스퀘어5 SP5 스튜디오의 Design 탭 예시]

STEP2. GridView 컬럼의 속성을 설정합니다.

[웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시]

<!-- gridView 의 소스 본문 예시 -->
<w2:gridView dataList="data:dlt_books">
    <!-- 중략 -->
    <w2:column copyOption="display" id="price" dataType="number" displayFormat="#,###">
    </w2:column>
    <!-- 중략 -->
</w2:gridView>
